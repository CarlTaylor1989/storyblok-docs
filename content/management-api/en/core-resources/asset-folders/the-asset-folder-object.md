| Property | Description |
|---|---|
| `id` | Numeric Unique ID |
| `name` | Display name of your asset folder |
| `parent_id` | Parent asset folder id |

;examplearea

Example Object
 
```json
{
  "asset_folder": {
    "id": 41,
    "name": "Commerce",
    "parent_id": null
  }
}
```
