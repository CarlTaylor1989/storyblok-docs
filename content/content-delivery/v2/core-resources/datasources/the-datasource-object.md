---
title: The Datasource Object
---

| Property            | Description          |
|---------------------|----------------------|
| `id`                | Numeric id | 
| `name`              | Given name  | 
| `slug`              | Given slug | 
| `dimensions`        | Array of dimension objects | 

;examplearea

Example Object

```json
{
  "id": 1433,
  "name": "Label",
  "slug": "labels",
  "dimensions": [
    {
      "id": 126,
      "entry_value": "en",
      "name": "English"
    }
  ]
}
```
