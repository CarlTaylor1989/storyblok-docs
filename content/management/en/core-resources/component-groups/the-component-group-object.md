---
title: The Component Group Object
---

| Property | Description |
|---|---|
| `id` | Numeric Unique ID |
| `name` | Name of the group |
| `uuid` | Uuid of the group |

;examplearea

Example Object

```json
{
  "component_group": {
    "id": 214123,
    "name": "Teasers",
    "uuid": "19323-32144-23423-42314"
  }
}
```
